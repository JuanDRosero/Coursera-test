Chicken

El pollo es una Comida muy deliciosa, existen diferetes formas de preparalo, entre las que puedo recordar se encuetra el pollo frito, el pollo crunch, el pollo broster, el pollo asado, el pollo ahumado. Adicionalmente es posible añadirlo en diferentes comidaas, mi preferida es el ajiaco con pollo, el cual es un plato tipico de Colombia.

Beef

La carne de res es un alimento muy nutritivo y delicioso, tiene varias formas de prepararase y existe una diversidad de platillos que la incluyen. Entre las diferentes formas de prepararla se encuentra la carne asada, la carne sudada la carne frita y la carne molida. Esta ultima es el aderezo perfecto a las pastas.

Sushi

El sushi se prepara generalmente en raciones pequeñas, aproximadamente del tamaño de un bocado, y puede adoptar diversas formas. Si se sirven el pescado y el arroz enrollados en una hoja de alga nori se le llama maki (‘rollo’). Si se trata de una especie de albóndiga de arroz cubierta por el pescado hablamos de nigiri.
